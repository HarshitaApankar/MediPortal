# MediPortal
A healthcare management system built using Python, HTML, CSS and PostgreSQL.
**MediPortal** is a healthcare web application developed using Python, Flask, HTML, CSS, and PostgreSQL. The platform helps users find hospitals, register as blood donors, submit blood requests, book appointments, and access healthcare-related information through an easy-to-use interface. It integrates a PostgreSQL database to efficiently manage user, donor, hospital, and appointment data, providing a centralized solution for connecting patients with healthcare services and blood donation support.
